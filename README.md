# Security Operations Labs & Threat Intel Notes

This repository is a collection of notes, playbooks, and walkthroughs related to Security Operations (SOC), Threat Intelligence, and Incident Response. The goal is to provide practical, real-world examples and summaries of key concepts for security professionals.

This is a living repository and will be updated with new content, techniques, and scenarios.

## Contents

-   **[MITRE ATT&CK Summaries](./mitre-attack-techniques/)**: Quick summaries of common adversary techniques.
-   **[Incident Response Playbooks](./incident-response-playbooks/)**: Mock playbooks for responding to various security incidents.
-   **[Alert Triage Examples](./alert-triage-examples/)**: Walkthroughs of analyzing and triaging sample alerts and logs.
-   **[Phishing Analysis](./phishing-analysis/)**: Step-by-step guides for analyzing suspicious emails.
-   **[SOC Documentation](./soc-documentation/)**: Examples of documentation used in a SOC, such as risk registers and workflows.
-   **[Tool Workflows](./tool-workflows/)**: Conceptual guides on how to use common security tools like EDR, SIEMs, etc.
